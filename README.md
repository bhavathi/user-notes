## Available Scripts

In the project directory, you can run:

## `npm install / yarn`

Installs the required node modules.

## `npm run dev / yarn dev`

Runs the app in the development mode.<br>
Open [http://localhost:3003](http://localhost:3003) to view it in the browser.
